# Student Record System

A **command‑line Python application** that demonstrates all core concepts from **Module 1** of the Python course.

## Features
- Add, view, search, update, and delete student records
- Calculate statistics (average GPA, age, active/inactive counts, etc.)
- Filter students by GPA range, active status, subject, or age range
- Persistent storage using a JSON file (`students.json`)
- Robust input validation and error handling
- Use of **variables, data types, operators, control flow, loops, functions, strings, lists, tuples, dictionaries, sets, file I/O, and exception handling**.
